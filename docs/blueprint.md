# **App Name**: MoneyWise

## Core Features:

- Monthly Income Configuration: Section to register and edit the monthly income, including historical data.
- Real-time Budget Tracking: Display the total income and remaining balance in real-time.
- Expense Logging: Log expenses with description, price, and optional photo using a floating window.
- Generative Insights: The LLM provides users with personalized financial insights by acting as a tool, to recommend potential savings or investment strategies, based on income and expense patterns.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of calm and security.
- Background color: Very light grey (#F5F5F5) for a clean and modern look.
- Accent color: Pale blue (#ADD8E6) for interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and slightly warm feel.
- Prioritize a clean and intuitive layout. Income displayed in the header with green text, expense list in the middle, and remaining balance at the bottom.
- Use subtle, modern animations for transactions and updates.
